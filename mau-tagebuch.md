# Mau Tagebuch

Ab jetzt liegt jeder Tagebucheintrag in einer eigenen Datei unter:
/home/clawdi/.hermes/diary/entries/

Dateinamensschema:
- YYYY-MM-DD.md

Beispiel:
- 2026-04-04.md

Hinweise:
- Jede Datei enthält genau einen Tagebucheintrag für einen Tag.
- Das Template für neue Einträge steht in /home/clawdi/.hermes/diary/DIARY_TEMPLATE.md.
- Cover-Bilder können passend unter assets/diary-covers/<YYYY-MM-DD>.jpg referenziert werden.
- Slack-Versand liest künftig direkt die Tagesdatei.
